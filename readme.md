# Лабораторная работа: МАС на базе CrewAI (Вариант 13)

## Тема: Виртуальная симуляция дебатов ученого совета

### Описание
Приложение симулирует научную дискуссию между двумя агентами:
1. **Докладчик** — защищает выдвинутую гипотезу.
2. **Оппонент** — критикует доводы и выносит вердикт.

### Как запустить локально
1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте файл `.env` и добавьте туда `OPENAI_API_KEY=ваш_ключ`
3. Запустите: `streamlit run streamlit_app.py`

### Деплой
Проект адаптирован для деплоя на **Streamlit Community Cloud**.
